DESCRIPTION

r.idsm.import.bb downloads image based digital surface model (iDSM, in German bDOM) for Brandenburg (BB) and area of interest. The data "bildbasiertes Digitales Oberflächenmodell" is downloaded from OpenGeodata.Brandenburg and can be used under the specification of the licence: © GeoBasis-DE/LGB, Datenlizenz Deutschland - Namensnennung - Version 2.0, © GeoBasis Brandenburg/Bildbasiertes Digitales Oberflächenmodell (bDOM).

EXAMPLE

Import iDSM

Import iDSM with native resolution: 
r.idsm.import.bb aoi=aoi_BB output=idsm_BB -r
